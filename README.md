# Repozitorii

Учебный репозиторий по QA-автоматизации.

## Как запускать тесты

1. Создать и активировать виртуальное окружение:
   ```
   python -m venv venv
   venv\Scripts\Activate.ps1
   ```
2. Установить зависимости:
   ```
   pip install -r requirements.txt
   ```
3. Запустить тесты:
   ```
   pytest
   ```
